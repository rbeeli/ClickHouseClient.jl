# Benchmark Results

Generated by `benchmarks/run.jl`.

## Reproduce

Start ClickHouse with:

```sh
just start
```

Run the benchmark with normalized options:

```sh
julia --project=. benchmarks/run.jl \
  --host 'localhost' \
  --native-port '9000' \
  --http-port '8123' \
  --user 'default' \
  --database '' \
  --rows '100000' \
  --iterations '5' \
  --clients 'julia,python-connect,go,rust' \
  --output 'benchmarks/results/latest.json' \
  --markdown 'benchmarks/RESULTS.md'
```

Passwords are intentionally omitted from this generated command.

## Workloads

- `read_scan_materialize`: ordered scan and client-side materialization.
- `read_group_aggregate`: grouped aggregate over low-cardinality dimensions.
- `write_batch_insert`: typed batch insert into an in-memory table.
- `write_row_records`: row-record insert path into an in-memory table.
